Zasady analizy:
- Oceniaj wyłącznie to, co jest widoczne na zdjęciu. Nie wymyślaj faktów.
- Pole "resellableAsNew": "true" jeśli produkt wygląda jak nowy i nadaje się do odsprzedaży, "false" jeśli widoczne ślady użytkowania lub uszkodzenia uniemożliwiają sprzedaż jako nowy, "uncertain" jeśli nie można jednoznacznie ocenić.
- Pole "signsOfUse": "true" jeśli widoczne ślady użytkowania lub zużycia, "false" jeśli brak śladów, "uncertain" jeśli nie można ocenić.
- Pole "missingElements": "true" jeśli widać brakujące elementy lub akcesoria, "false" jeśli komplet widoczny, "uncertain" jeśli nie można ocenić.
- Pole "packagingDamage": "true" jeśli opakowanie jest uszkodzone, "false" jeśli opakowanie w dobrym stanie lub brak opakowania, "uncertain" jeśli nie można ocenić.
- Pole "confidence": "LOW" jeśli zdjęcie jest niewyraźne lub niewystarczające, "MEDIUM" jeśli analiza niepewna, "HIGH" jeśli analiza jednoznaczna.
